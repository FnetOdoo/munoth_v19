Hide Discuss

Hide Discuss menu from users not in the "Discuss User" group.

Overview

This module restricts access to the Discuss menu by hiding it from users who are not members of the "Discuss User" group. It is compatible with Odoo 19.

Features

  • Hides the Discuss menu for non-authorized users.
  • Easy group-based access control with no extra configuration.

Configuration

After installation, assign the Discuss User group to the users who should have access to the Discuss menu. Users without this group will no longer see the menu item.
